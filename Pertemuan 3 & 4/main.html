Tugas-3 Web Desain

Ini adalah area Main Content


Web design atau desain web adalah proses perencanaan dan pembuatan tampilan situs web.
Desain web meliputi struktur, tata letak, warna, gambar, font, dan grafik.


Tujuan web desain adalah:

  • Menciptakan tampilan dan nuansa keseluruhan situs web
  • Memecahkan masalah komunikasi antara pengguna dan informasi halaman web
  • Membuat atau menyampaikan informasi secara cepat dan real-time
  • Memberikan kenyamanan tambahan pada pembaca atau pengunjung website
